# **App Name**: Chicken Bites Menu Viewer

## Core Features:

- Menu Display: Displays the menu items (Chicken popcorn, Chicken wings, Chicken lollipop, Chicken strips, Chicken pakodi, French fries, Perry perry French fries) with their prices (Single and Full) from the menu.
- Image Integration: Incorporates attractive food photography for each menu item, focusing on graphical representation to make the site visually appealing.
- Contact Information: Displays contact information (phone number: +91 8331988366) for orders, making it easily accessible for users.

## Style Guidelines:

- Primary color: Saturated golden yellow (#E4C377), reminiscent of fried chicken, creating an appetizing visual.
- Background color: Desaturated brown (#382B22), mimicking the menu's original background.
- Accent color: Pale gold (#B4A477) for subtle highlights and dividers.
- Headline font: 'Belleza', a humanist sans-serif that matches the hand-crafted feel of the food. Body font: 'Alegreya' (serif) for longer food descriptions.
- Use minimalist line icons for each menu category to maintain a clean and modern interface.
- Grid-based layout with ample spacing to create a visually appealing and easily scannable menu.
- Subtle hover animations on menu items to provide feedback and improve user engagement.
